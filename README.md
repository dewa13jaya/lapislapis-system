# LapisLapis Production & Sales System

Sistem manajemen stok dan order untuk tim produksi dan sales LapisLapis.

## Fitur
- Dashboard realtime
- Manajemen stok (masuk, keluar, retur)
- Order management dengan flow status
- Cetak surat jalan
- Master produk & outlet

## Tech Stack
- React 18
- Supabase (database + realtime)
- Vercel (hosting)

## Setup
1. Clone repo ini
2. `npm install`
3. `npm start`
